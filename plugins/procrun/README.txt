How to re-create geronimosrv.exe and geronimosrvw.exe
-----------------------------------------------------

To recreate the geronimosrv.exe and geronimosrvw.exe, you just need to
download Apache Commons Daemon 1.0.2 release for Windows from 
http://www.apache.org/dist/commons/daemon/binaries/1.0.2/windows/commons-daemon-1.0.2-bin-windows.zip

Unzip the downloaded file and rename prunsrv.exe to geronimosrv.exe, and
rename prunmgr.exe to geronimosrvw.exe.
